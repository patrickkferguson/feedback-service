AAA Feedback Widget

Usage

Include the widget script by adding the following HTML to your web page source (usually inside the head tag): 


    <script src="http://aaafeedback-1494464907.ap-southeast-2.elb.amazonaws.com/feedback-widget.v1.js"></script>

You will also need to define a unique page id for each page. The page id should be a standard GUID and should be declared as a variable named aaa_feedback_pageId. Here is an example: 


    <script>
        function uuidv4() {
            return ([1e7]+-1e3+-4e3+-8e3+-1e11).replace(/[018]/g, c =>
                (c ^ crypto.getRandomValues(new Uint8Array(1))[0] & 15 >> c / 4).toString(16)
            )
        }
        const aaa_feedback_pageId = uuidv4();
    </script>
